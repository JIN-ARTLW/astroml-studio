# v0 사용자 흐름 (분류, 화면 단위)

← [캔버스](../CANVAS.md)

목표: **비전공 학생이 도움 없이 ≤ 15분에 분류기 완성** (가설 H1).

## 화면 흐름

```
1. 새 프로젝트        2. 데이터          3. 라벨링
   작업=분류 선택  →   데모 불러오기  →   클래스 클릭
   클래스 정의         or 업로드          (or 폴더명 자동)
        │                                      │
        ▼                                      ▼
6. 예측            5. 결과            4. 학습(브라우저)
   새 이미지 →    혼동행렬·곡선  ←   백본 선택·Start
   클래스+신뢰도   오분류 갤러리       라이브 정확도
        │
        ▼
   (선택) "Colab에서 본격 학습" → 노트북 생성
```

## 화면별 상세

### 1. 새 프로젝트
- 작업 유형 선택(현재 활성: **분류**, 나머지는 "곧 제공" 비활성).
- 클래스 정의: 이름 입력(예: smooth-round / edge-on-disk / spiral).

### 2. 데이터
- **데모 불러오기** 버튼 → [Galaxy10 3클래스 서브셋](datasets.md) 자동 로드(가장 빠른 첫 성공).
- 또는 드래그&드롭 업로드(PNG/JPG). 폴더별 업로드 시 폴더명=클래스 자동 매핑.

### 3. 라벨링
- 그리드 갤러리 + 클래스 버튼(숫자키 1/2/3 단축키).
- 데모/폴더명 사용 시 자동 라벨 → 라벨링 단계 건너뜀.
- 진행률 표시(라벨 N/전체).

### 4. 학습 (브라우저)
- 백본 선택: MobileNetV2(기본, 빠름).
- 분할: train/val 자동(예: 80/20).
- Start → 임베딩 추출 → Dense 헤드 학습. **에폭별 정확도/손실 라이브 차트.**
- 전부 클라이언트 사이드(서버비 0).

### 5. 결과
- 혼동행렬, per-class precision/recall.
- 학습/검증 곡선.
- 오분류 갤러리(틀린 이미지 + 예측·정답).

### 6. 예측 / 내보내기
- 새 이미지 드롭 → 클래스 + 신뢰도(top-k).
- 내보내기: TF.js 모델(json+weights), 라벨/지표 CSV.
- **"Colab에서 본격 학습"** → [노트북 자동생성](compute-hybrid.md).

## v0 완료 정의 (Definition of Done)
- [ ] 데모 데이터로 1~6 단계 끊김 없이 완주
- [ ] 3클래스 정확도 ≥ 0.8 (H2)
- [ ] 코드 0줄, 설치 0
- [ ] 모바일/데스크톱 브라우저에서 동작
