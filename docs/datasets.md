## v0 데모 (분류) — 확정

### 1순위: Galaxy10 DECaLS
- 출처: `astroNN` 프로젝트 (ML 튜토리얼용으로 설계됨 → 진입장벽 최저).
- 규모: 약 **17,736장**, **256×256×3 RGB**, **10 클래스** (형태별).
- 라벨: Galaxy Zoo 투표 기반.
- 다운로드: 단일 `.h5` 파일 (HDF5). 브라우저 데모는 일부만 추출해 PNG로 변환.
- 라이선스: 공개.

### v0 단순화 셋 (브라우저 전이학습용)
10클래스 전부는 브라우저 전이학습엔 무겁다 → **시각적으로 뚜렷한 3클래스로 서브셋**:
- `smooth-round` (둥근 매끈)
- `edge-on-disk` (측면 원반)
- `spiral` (나선)
- 클래스당 ~300~500장이면 H2(정확도 ≥ 0.8) 검증 가능.

### 대안: Galaxy10 SDSS
- 약 21,785장, 69×69, 10클래스. 더 가볍지만 해상도 낮음. 브라우저 속도 우선이면 후보.
