# AstroML Studio

천문·우주과학 이미지에서 **머신러닝을 몰라도** 원하는 결과(분류·검출·물리량·이상)를 얻게 해주는 노코드 연구툴.
대상: 천문/물리 학부 고학년 ~ 대학원생.

- 연구 캔버스(설계의 토대): [CANVAS.md](CANVAS.md)
- 상태: **v0 구상** — 분류(갤럭시 형태) + 브라우저 전이학습부터.
- 컴퓨팅: 하이브리드(브라우저 전이학습 + Colab 자동 노트북).
- 무료 구조: Vercel + Supabase + 브라우저 ML + 사용자 무료 GPU.

> 코드명/리포명은 변경 가능.
